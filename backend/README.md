# Backend express API CMsysAdmin

This project was generated using [Express](https://expressjs.com/en/starter/installing.html) version 5.1.

## Development server

To start a local development server, run:

```bash
console-ninja node --watch app.js
```

Once the server is running, open postman and use url base `http://localhost:8800/api/` , and routes to test API. The application will automatically reload whenever you modify any of the source files.

- auth -> autorizations, login, logout
- users -> user crud
